## 실행 방법 (프로토타입)

### 사전 설치
- Node.js 20+ (https://nodejs.org)

### 설치
```
npm install
```

### 개발 서버 실행
```
npm run dev
```

브라우저로 표시된 `Local` URL 접속. 같은 Wi-Fi의 폰에선 `Network` URL로 접속.

### 테스트
```
npm run test
```

### 빌드 (정적 파일 생성)
```
npm run build
```

`dist/` 폴더에 결과물.
